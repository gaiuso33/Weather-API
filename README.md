# Weather App ( with a 3-Day Forecast)

A simple Weather App that allows users to search for any city in the world and view:

- Current weather conditions
- Temperature (Celsius / Fahrenheit toggle)
- Feels-like temperature
- Humidity and wind speed
- 3-day weather forecast
- Dynamic background changes depending on the weather condition

This project uses **WeatherAPI.com** for weather data, and includes a **Node.js + Express backend** to securely hide the API key.

---

## Features

- Search weather by city name  
- Shows current temperature + condition  
- Shows 3-day forecast  
- Toggle between °C and °F  
- Loading indicator while fetching data  
- Weather icons displayed dynamically  
- Background changes depending on weather (sunny, cloudy, rain, storm, etc.)  
- API key is hidden using `.env` file
